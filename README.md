# Developer's app

### Requisitos
- Docker;
- VS Code;
- Terminal WSL

### Instalação
- Clonar o projeto e executar o comando **$ ./_start.sh**

### Aplicação
- Acessar a URL da aplicação **http://localhost/developers**

### Testes
- Executar o comando **$ docker-compose exec app php artisan test**
